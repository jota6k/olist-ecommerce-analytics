## Project Overview
This project analyzes Brazilian e-commerce data from Olist to extract business insights.

## Business Questions
- How does revenue evolve over time?
- Which states generate more sales?
- What is the average delivery delay?
- How do reviews relate to delivery performance?

## Tech Stack
- Excel
- Python
- SQL
- Power BI
- AWS

## Project Structure
- data/raw  
  Raw CSV files from the original Olist dataset. These files are not modified.

- data/processed  
  Cleaned and transformed datasets generated during the data processing stage.

- notebooks  
  Jupyter notebooks used for exploratory data analysis and feature engineering.

- sql  
  SQL scripts containing table creation and business-oriented queries.

- powerbi  
  Power BI dashboard files and documentation.

- docs  
  Project documentation, including architecture and design decisions.

## Data Processing

Initial data inspection and validation were performed using Excel.
Data cleaning and transformations are documented and later reproduced in Python.
