## Processed Data

This folder contains datasets derived from the raw Olist data.

### Processing Steps (Excel)
- Date formats standardized across all date-related columns.
- City names standardized to ensure consistency.
- Monetary values standardized and validated.
- Missing values identified and reviewed.
- Key relationships validated (order_id, customer_id).

Excel was used only for initial inspection and basic validation.
All analytical transformations are performed using Python and SQL.
